Добро пожаловать на РМК-2025.

## Что за билд?
Это - хардфорк [Space Station 14](https://github.com/space-wizards/space-station-14), пытающийся воссоздать механики экстракшен-шутеров, при этом совмещая их с РП аспектом игры. 

## Какие у вас основные механики?
Сейчас - никаких.
Проект только начался, очнитесь.

А вообще, в планах у нас перенос части механик из экстракшен-шутеров, реворк медицины ЕЕ и создание первого полноценного геймплей-цикла, который при этом будет самодостаточным для ранних тестов.
